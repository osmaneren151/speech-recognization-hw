\# Konuşma Tanıma Ödevi



Bu proje, Hidden Markov Model (HMM) kullanılarak basit bir konuşma tanıma örneğini gerçekleştirmektedir.



\## İçerik

\- EV kelimesi için teorik Viterbi çözümü

\- EV ve OKUL için oyuncak HMM modelleri

\- Yeni bir gözlem dizisinin log-likelihood skorları ile sınıflandırılması



\## Kullanılan Teknolojiler

\- Python 3.12

\- numpy

\- hmmlearn

\- scikit-learn



\## Çalıştırma

```bash

py -3.12 -m pip install -r requirements.txt

py -3.12 recognizer.py
